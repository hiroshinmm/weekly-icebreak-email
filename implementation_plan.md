# 実装計画: 毎週のエンジニア向けアイスブレイク自動配信

## 概要
毎週、最新のITニュースからエンジニアが盛り上がりそうなトピックを1つ厳選し、見栄えの良い「スライド画像（A4横）」を自動生成してメールで送りつけるシステムを構築します。

## 構成
1.  **データ取得**: Zenn または はてなブックマークのトップ記事を1つ取得。
2.  **画像生成**: `puppeteer` を使い、動的に生成したHTML（モダンなスライドデザイン）を A4横サイズ (1414x1000px) の PNG 画像としてキャプチャ。
3.  **メール配信**: `nodemailer` を使用。
    - ※SMTPサーバの設定が必要です（GmailやOutlookなどの情報を設定ファイルに記述）。
4.  **自動化**: Windows タスクスケジューラで毎週特定の日時に実行。

## 検討事項
- **メールの設定**: SMTPのホスト、ポート、ユーザー名、パスワードを用意する必要があります。
- **デザイン**: ソフトウェアエンジニアが好むダークモードや、清潔感のあるプレミアムなデザインをHTML/CSSで構築します。

## 確認方法
- `node index.js` を実行し、デスクトップに画像が生成され、自分宛にメールが届くことを確認。
