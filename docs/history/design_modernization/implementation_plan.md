# 実装計画: デザインのモダン化と視認性向上

## 目的
現在のレイアウトを維持しつつ、視覚的なプレミアム感と情報の読みやすさを大幅に向上させます。

## デザインの改善方針

### 1. タイポグラフィと配色
- フォント: システムフォントの最適化に加え、読みやすい行間 (line-height) を設定。
- 配色: ソニーらしい信頼感のあるネイビー (`#003399`) をアクセントにしつつ、背景には目に優しいオフホワイトや淡いグレーを使用。

### 2. コンポーネントのブラッシュアップ
- **カード**: 角丸 (`border-radius: 24px`) や繊細なドロップシャドウを使用し、立体感を演出。
- **インサイトボックス**: 背景色をより淡くし、境界線を柔らかくすることで、本文との調和を図りつつ強調。
- **タグ**: 視認性を保ちつつ、主張しすぎない洗練されたデザインに変更。

### 3. Web ギャラリーのUX向上
- カードのホバー時に浮き上がるアニメーションを追加。
- 画面幅に応じた柔軟なレスポンシブレイアウト。

## 変更内容
- `src/templateGenerator.js` 内の `generateEmailTemplate` および `generateIndexHtml` のスタイル定義を刷新します。

## 検証計画
- `node index.js` を実行し、生成される `docs/index.html` をブラウザで確認。
- 送信されるメールの表示をシミュレートし、主要なメールクライアントで見栄えが良いか確認。
