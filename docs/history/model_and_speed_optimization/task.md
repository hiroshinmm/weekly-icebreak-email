- [ ] `index.js` の考察生成ループにおける待機時間を最適化し、全体の実行時間を削減する。

## 完了条件
- `npm start` 相当の処理（またはテスト実行）が正常に完了すること。
- 各カテゴリのニュースが取得でき、それらに対してAI考察が付与されること。
- 並列化により実効速度が改善されていること。
