# 実装計画: レイアウト変更と要約の調整

## 目的
ユーザーのフィードバックに基づき、情報の表示順序を最適化し、ニュースの要約文をより詳細な内容に変更します。

## 変更内容

### 1. 要約の長尺化 (`src/insightGenerator.js`)
- Gemini へのプロンプトで、`translatedSnippet` の文字数制限を「100〜150文字」から「200〜250文字程度」に引き上げます。

### 2. レイアウト順序の変更 (`src/templateGenerator.js`)
- メールおよび Web ページの表示順序を以下の通りに変更します：
    1.  タイトル (Title)
    2.  画像 (Image)
    3.  要約 (Summary/Snippet)
    4.  インサイト (AI Insight)

## 検証計画
- `node index.js` を実行し、生成されるメール本文および `index.html` の表示順序が正しく、要約が長くなっていることを確認します。
