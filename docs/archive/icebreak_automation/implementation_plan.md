# 実装計画: 毎週のエンジニア向けアイスブレイク自動配信

> [!IMPORTANT]
> このプロジェクトは、毎週月曜日に最新の技術トレンドを1件抽出し、視覚的なスライド画像としてエンジニアにメール配信することを目的としています。

## 概要
最新のITニュースからエンジニアが盛り上がりそうなトピックを1つ厳選し、見栄えの良い「スライド画像（A4横）」を自動生成してメールで送りつけるシステムを構築します。

## システム構成図

```mermaid
graph TD
    A[RSSフィード (Zenn/はてブ)] -->|データ取得| B(Node.js スクリプト)
    B -->|URLのQRコード化| C{スライド生成}
    B -->|HTML/CSS レンダリング| C
    C -->|Puppeteer キャプチャ| D[A4 スライド画像 (PNG)]
    D -->|Nodemailer 添付| E[Gmail 送信]
    F[Windows タスクスケジューラ] -->|毎週定期実行| B
```

## 構成コンポーネント

### 1. データ取得モジュール
- **ソース**: Zenn (トレンド) または はてなブックマーク (ITカテゴリ)
- **手法**: `rss-parser` を使用し、最新の1件をピックアップ。

### 2. ビジュアル生成エンジン
- `puppeteer` を駆動し、専用のHTMLテンプレートをレンダリング。
- **デザイン**: 清潔感のあるダークモード、プレミアムなタイポグラフィ（Inter/Noto Sans JP）。
- **新機能**: 記事へ直接アクセスできる **QRコード** を右下に埋め込み。
- **形式**: A4 Landscape (1414x1000px) の高解像度PNG。

### 3. 自動配信レイヤー
- `nodemailer` を使用し、Gmail経由で画像を配信。
- `config.json` によるセキュアな設定管理。

## 安全性と運用

> [!TIP]
> Gmailを使用する場合、通常のパスワードではなく「アプリパスワード」を生成して使用することで、安全性を確保します。
